## What each transformation does

For every `src/__tests__/{get,make,put}_verb_page/{particle}.test.tsx`:

1. **Add import** after `from './helpers'` line:
   ```ts
   import { describeChevronAndColour, describeSectionToggle } from '../verbPage/sharedSectionTests';
   ```
2. **Add `const LABEL`** before `beforeEach`:
   ```ts
   const LABEL = 'GetVerbPage'; // or Make/Put
   ```
3. **Remove** the entire `describe('... section toggle', ...)` block.
   - If it contained an `it('...does not affect...', ...)` test, extract it into a new block:
     ```ts
     describe(`${LABEL} — "particle" section independence`, () => { /* extracted test */ });
     ```
4. **Replace** with factory call:
   ```ts
   describeSectionToggle(LABEL, 'particle', 'storageKey', /firstDef/i, renderPage);
   ```
5. **Remove** the entire `describe('... chevron and colour', ...)` block.
6. **Replace** with factory call:
   ```ts
   describeChevronAndColour(LABEL, 'particle', renderPage);
   ```
7. All other `describe` blocks (definitions, card expand, localStorage, etc.) stay unchanged.

## Implementation approach

Write a Python script `/tmp/transform_tests.py` that:
1. Reads each file
2. Locates toggle describe block (`find_top_describe` by keyword `'section toggle'`)
3. Extracts any `it('...does not affect...',` blocks within toggle block
4. Locates chevron describe block (`find_top_describe` by keyword `'chevron and colour'`)
5. Rebuilds file: header (with new import + LABEL) → `describeSectionToggle(...)` → optional independence describe → `describeChevronAndColour(...)` → unchanged remainder

All per-file data (label, particle, storageKey, firstDef) is hardcoded in the script from the full table compiled in the previous session.
