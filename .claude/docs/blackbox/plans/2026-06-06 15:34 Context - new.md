# Context

Clicking a phrasal verb card in the list causes a visible layout shift: the card grows taller and internal spacing changes. This happens because the collapsed and expanded states use *entirely different DOM structures* with different padding values and margin placements, even when no image is shown.

## Root Cause

In `src/components/VerbPage.tsx`, the `Meaning` component renders two completely different JSX trees:

**Collapsed** (`p-4` padding, `mt-0.5` gap):
```
<div class="p-4">
  <div class="flex items-start gap-2">
    <span>[badge]</span>
    <div class="min-w-0 flex-1">
      <p class="... truncate">[definition]</p>
      <p class="... truncate mt-0.5">[example]</p>
    </div>
  </div>
</div>
```

**Expanded** (`p-4 pt-3` padding, `mb-2` gap, `pl-8` on example):
```
<div class="p-4 pt-3">
  [optional image]
  <div class="flex items-start gap-2 mb-2">
    <span>[badge]</span>
    <p>[definition]</p>
  </div>
  <p class="pl-8">[example]</p>
</div>
```

Differences that shift the layout even without an image:
- Top padding: `16px` (collapsed) vs `12px` (expanded)
- Definition→example gap: `mt-0.5` / 2px (collapsed) vs `mb-2` / 8px (expanded)
- Example alignment: inside `flex-1` div (collapsed) vs `pl-8` standalone (expanded)

## Fix

Replace the ternary with a single unified layout. Only the `truncate` classes and image visibility change based on state.

**File:** `src/components/VerbPage.tsx`, `Meaning` component (lines 39–96)

```jsx
<div
  className="bg-white dark:bg-gray-900 rounded-2xl border border-gray-600 dark:border-gray-400 shadow-md overflow-hidden cursor-pointer"
  onClick={toggle}
>
  <div className="p-4">
    {!collapsed && !isDefault && (
      <div className="flex justify-center mb-3">
        <img src={imageSrc} alt={imageAlt}
          className="max-w-sm w-full border border-gray-200 dark:border-gray-700 rounded-xl" />
      </div>
    )}
    <div className="flex items-start gap-2">
      <span className="flex-shrink-0 w-6 h-6 rounded-full bg-blue-600 text-white text-xs font-bold flex items-center justify-center">
        {number}
      </span>
      <div className="min-w-0 flex-1">
        <p
          className={`text-gray-800 dark:text-gray-100 text-xl md:text-base font-medium leading-snug${collapsed ? ' truncate' : ''}`}
          title={definition}
        >
          {definition}
        </p>
        <p
          className={`text-gray-500 dark:text-gray-400 text-xl md:text-sm italic mt-0.5${collapsed ? ' truncate' : ''}`}
          title={`"${example}"`}
        >
          "{example}"
        </p>
      </div>
    </div>
  </div>
</div>
```

This eliminates the layout shift: padding and spacing are identical in both states; the only change is `truncate` being removed (allowing long text to wrap) and the image appearing.
